# TrendPulse 

TrendPulse is a Python-based data pipeline that fetches, processes, analyzes, and visualizes trending stories from the Hacker News API.

##  Project Structure
- **Task 1:** Fetch trending data using the Hacker News API
- **Task 2:** Clean and convert JSON data into CSV format
- **Task 3:** Analyze trends using Pandas and NumPy
- **Task 4:** Visualize insights using Matplotlib and Seaborn

## Technologies Used
- Python
- Requests
- Pandas
- NumPy
- Matplotlib
- Seaborn
